# PT Mert

Mobil uygulama: özel ders veren eğitmenlerin müşterilerini, ödemelerini ve randevularını yönetmesi için geliştirilmiş bir sistemdir.

## Teknolojiler
- Flutter
- Firebase
- GetX
- Custom UI Theme

## Özellikler
- Giriş/Kayıt sistemi
- Randevu planlama
- Ödeme ve bakiye görüntüleme
- Modern UI bileşenleri
